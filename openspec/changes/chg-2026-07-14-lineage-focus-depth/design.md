# Design: 血缘实体穿透高亮

## 邻域计算

选中实体后，以当前经过实体和关系类型筛选的图为输入执行广度优先遍历。边按上下游双向可达处理：深度 1 包含所有直接入边和出边；深度 N 包含从选中实体经过不超过 N 条边可达的节点和遍历关系。已被用户隐藏的实体或关系不会参与高亮。

无选中实体时正常显示全部关系。存在选中实体时：

- 选中节点和邻域节点保留类型颜色。
- 邻域外节点降为浅灰，作为位置上下文。
- 只显示邻域中的关系和方向箭头。
- 邻域外名称标签隐藏。
- 点击画布空白或关闭详情面板时恢复完整图。

## 穿透加载

穿透层级限制为 `1-5`，默认 1。点击实体或调整层级时，通过现有单节点一跳 API 按广度逐层加载。单次穿透最多请求 40 个尚未展开的节点，并继续使用 500 节点合并上限；达到 API 截断、请求上限或节点上限时显示提示，但已加载邻域仍可交互。

加载过程复用节点加载状态，避免同一节点重复请求。新的选中、搜索、恢复任务骨架或切换项目会取消旧加载，防止迟到响应覆盖当前图。

## 交互

实体详情面板显示当前穿透层级及高亮实体/关系数量，并提供减号和加号步进按钮。选中节点适当放大，关联节点保留类型色，无关节点使用中性灰。关系标签只为当前关联边生成。

## 验证

- 单元测试覆盖一层、两层、无关连通分量和无效选中实体。
- 异步测试覆盖逐层请求、请求预算、节点上限和取消边界。
- Playwright 验证节点点击、层级步进、关系收敛、空白恢复和桌面/移动端非空画布。
