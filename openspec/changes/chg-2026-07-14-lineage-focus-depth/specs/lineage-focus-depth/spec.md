# 血缘实体穿透高亮规范

## ADDED Requirements

### Requirement: 点击实体必须只高亮关联链路

存在选中实体时，画布 MUST 只显示该实体在当前穿透层级内直接或间接可达的关系；关联节点 MUST 保留类型颜色，无关节点 MUST 弱化且无关标签 MUST 隐藏。

#### Scenario: 查看一层直接关系

- **WHEN** 用户选中实体 B 且穿透层级为 1
- **THEN** 只高亮与 B 直接相连的入边、出边和相邻实体
- **AND** 两跳以外关系不显示

#### Scenario: 增加穿透层级

- **WHEN** 用户把穿透层级从 1 调整为 2
- **THEN** 系统按需加载并高亮从选中实体两跳可达的上下游链路
- **AND** 不相关连通分量仍不显示关系

### Requirement: 穿透必须遵守现有类型筛选

邻域计算 MUST 以当前实体和关系类型筛选后的图为输入，MUST NOT 重新显示用户已经隐藏的类型。

#### Scenario: 字段关系已关闭

- **WHEN** 用户关闭字段节点后选中一张表并提高穿透层级
- **THEN** 高亮结果不包含字段节点或字段关系

### Requirement: 穿透必须受资源上限约束

穿透层级 MUST 限制在 1 至 5，单次穿透加载 MUST 遵循请求预算和 500 节点上限；达到上限时 MUST 保留已加载结果并提示用户。

#### Scenario: 高度连接的实体

- **WHEN** 多层穿透将超过请求预算或节点上限
- **THEN** 系统停止继续加载
- **AND** 当前已加载邻域保持可用

### Requirement: 取消选择必须恢复完整图

点击画布空白、关闭实体面板、搜索或恢复任务骨架 MUST 清除当前高亮范围，并按现有实体与关系筛选恢复图谱。

#### Scenario: 点击画布空白

- **WHEN** 用户在实体高亮状态点击画布空白
- **THEN** 全部当前筛选允许的关系重新显示
